## Objective

The objective of this task is to create a client-ready Power BI dashboard that
provides clear insights into sales performance across platforms, categories,
products, and cities using the provided raw sales dataset.

## Dashboard Features

The Power BI dashboard includes the following visuals and KPIs:

- MTD Sales and Previous Month Sales shown as KPI cards (platform-wise)
- Average Selling Price (ASP) at platform level
- Platform-level returns
- Net sales contribution by:
  - Platform
  - Category
  - Product

## Additional Enhancements

- Interactive filters for platform, category, and city
- Drill-down functionality for detailed product-level analysis
- Clear layout and labeling for easy business understanding

## Approach

- Cleaned and prepared raw data in Power BI
- Created calculated measures using DAX
- Used appropriate visualizations to represent key business metrics
- Focused on simplicity and clarity for end users

## Submission Details

- Dashboard is submitted as a Power BI (.pbix) file
- All visuals are organized in a single, easy-to-navigate report page

## Notes

- Dashboard is designed for business and stakeholder use
- Visuals can be extended further based on business requirements
